Free Roam

A Travel Agency themed blog; built with Next.js, SCSS & Strapi. GraphQL is used to query the Strapi CMS for data.

Features include:

  • Custom content-types built with Strapi - makes it easy to create & manage Holiday Packages, Blog Articles & About page with minimal coding knowledge.
  • Pagination of Blog Articles
  • Rendering strategies used:
    • SWR library for client-side data fetching & caching
    • Static Site Generation - improves load times by pre-rendering HTML with data
    • Incremental Static Regeneration - revalidates cached static data, ensuring updates to data on Strapi server are synced with the view on refresh
